# Billigfylla. A Next JS web app helping you get whasted for cheap

## Requirements
* `NPM` or `Yarn`
* `Node 14 or newer`


## How to run
```bash
git clone git@github.com:sebaglen/billigfylla.git
cd billigfylla

yarn install
# or
npm install

yarn dev
# or
npm run dev
```
